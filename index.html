Apa yang ada di proyek ini?

Dataset produk Amazon, notebook analisis hybrid recommender, aplikasi Flask untuk pencarian dan rekomendasi, serta front-end modern dengan tampilan responsif.

Content-Based

TF-IDF pada konten produk dan review untuk menangkap kemiripan berbasis teks.

Collaborative Filtering

Item-based similarity dari pola rating user untuk menangkap perilaku kolektif.

Catatan GitHub Pages

GitHub Pages tidak mengeksekusi Flask. Jadi yang bisa ditampilkan di sini hanya halaman statis. Jika ingin aplikasi interaktif penuh, gunakan hosting backend seperti Render, Railway, atau Azure App Service.
